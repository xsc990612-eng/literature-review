### 理想技术方案：情感分析(三分类) + Qwen3微调 + 推理优化

基于以上资源，推荐完整技术栈：

#### 阶段1: 微调（情感分析三分类）
```bash
# 使用LLaMA-Factory微调Qwen3-8B
llamafactory-cli train \
    --model_name_or_path Qwen/Qwen3-8B \
    --dataset sentiment_3class \
    --template qwen3 \
    --finetuning_type lora \
    --lora_target q_proj,v_proj \
    --lora_rank 8 \
    --lora_alpha 32 \
    --learning_rate 2e-5 \
    --num_train_epochs 3 \
    --per_device_train_batch_size 2 \
    --gradient_accumulation_steps 8 \
    --fp16
```

#### 阶段2: 导出优化模型
```bash
# 合并LoRA权重
llamafactory-cli export \
    --model_name_or_path Qwen/Qwen3-8B \
    --adapter_name_or_path ./qwen3-sentiment-lora \
    --template qwen3 \
    --export_dir ./qwen3-sentiment-merged
```

#### 阶段3: 推理部署优化
```bash
# 方案A: vLLM部署（推荐）
vllm serve ./qwen3-sentiment-merged \
    --tensor-parallel-size 1 \
    --max-model-len 4096 \
    --quantization fp8  # 可选量化

# 方案B: TensorRT-LLM部署（极致性能）
trtllm-build \
    --checkpoint_dir ./qwen3-sentiment-merged \
    --output_dir ./qwen3-sentiment-trt \
    --quantization int8_weight_only
```

### 6.2 推理代码
```python
from transformers import AutoTokenizer, AutoModelForCausalLM
import torch

model_path = "./qwen3-sentiment-merged"
tokenizer = AutoTokenizer.from_pretrained(model_path, trust_remote_code=True)
model = AutoModelForCausalLM.from_pretrained(
    model_path,
    torch_dtype=torch.bfloat16,
    device_map="auto",
    trust_remote_code=True
)

def predict_sentiment(text):
    prompt = f"请判断以下文本的情感倾向(正面/中性/负面)：\n\n{text}\n\n情感："
    inputs = tokenizer(prompt, return_tensors="pt").to(model.device)
    outputs = model.generate(**inputs, max_new_tokens=5, do_sample=False)
    result = tokenizer.decode(outputs[0], skip_special_tokens=True)
    return result.split("情感：")[-1].strip()
```
